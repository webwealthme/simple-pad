    Simple Pad

    Simple Pad is a notepad to store notes, texts or even poems. It is focused in simplicity and speed for quick note taking while inspiration is at its peak.

    • Customizable: Choose font color, size and more (top left)
    • 5 Pages: Use them to save different content (bottom left)
    • Saves as you type: If you close the browser it will still be there

    Simple Pad is build in HTML5/jQuery and it is a great example on how modern websites can be so tiny yet so functional. You can create your own themes very easily. You can also use OmPad while being offline.
